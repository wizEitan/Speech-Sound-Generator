# Speech Sound Generator

**Speech Sound Generator** is a desktop application for creating high-quality WAV audio files — quickly and easily.

## Features

- 🎤 **Text-to-Speech** – Convert any text into a WAV audio file
- 🎙️ **Direct Recording** – Record audio directly from your microphone
- 📂 **Import & Convert** – Import audio from other formats and export as WAV

## Getting Started

1. Download the latest version from the [Releases](../../releases) page
2. Run the executable
3. Choose your input method: type text, record, or import a file
4. Export your WAV file

## System Requirements

- Windows 10 or Windows 11
